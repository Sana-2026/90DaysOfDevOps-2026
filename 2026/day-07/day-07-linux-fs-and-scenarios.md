üîπ Additional Directories

/bin

What it contains:
Essential command binaries required for system boot and repair.

Example (ls -l /bin):
ls, cp, mv, cat, bash

I would use this when ?
I run basic Linux commands even in minimal or recovery mode.

/usr/bin

What it contains:
Most user-level command binaries and applications.

Example (ls -l /usr/bin):
git, python3, vim, docker

I would use this when ?
I install and run standard user applications and tools.

/opt

What it contains:
Optional or third-party software installed manually.

Example (ls -l /opt):
google/, custom-app/

I would use this when ?
I install custom or third-party applications outside system defaults.
