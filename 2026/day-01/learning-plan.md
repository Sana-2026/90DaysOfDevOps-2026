# 90-Day DevOps Learning Plan

## Current Level
Experienced professional currently learning DevOps tools and cloud technologies with a focus on building practical skills.

## 90-Day Goals
1. Build and deploy a **containerized application using Docker and Docker Compose**.  
2. Create a **CI/CD pipeline using GitHub Actions or Jenkins** to automate build and testing.  
3. Deploy a **production-grade application on Kubernetes** using cloud infrastructure.

## Core DevOps Skills to Develop
1. **Linux troubleshooting and system fundamentals**  
2. **CI/CD pipeline design and automation**  
3. **Containerization and Kubernetes debugging**

## Weekly Time Commitment
- **Weekdays:** 2–2.5 hours per day  
- **Weekends:** 4–6 hours for hands-on practice and projects  

➡ Focus on **consistent daily practice, hands-on labs, and documenting learning on GitHub.**
